## 🔄 3회차 — 자산 가격 분석 [진행 중]

**목표**: 각 자산의 가격 흐름을 다양한 각도에서 시각화하고 비교한다.

**구현 내용**

1. **정규화 가격 차트** — 시작일 기준 100으로 맞춰 자산 간 성장률 직접 비교
   ```python
   normalized = price / price.iloc[0] * 100
   ```

2. **이동평균선 추가** — 단기(50일) vs 장기(200일) 추세 파악
   ```python
   ma50 = price.rolling(50).mean()
   ma200 = price.rolling(200).mean()
   ```

3. **상관관계 히트맵** — 자산 간 동조화 여부 확인 (seaborn heatmap)
   ```python
   price.corr()
   ```

4. **연도별 연간 수익률 막대 그래프** — 각 자산의 연도별 성과 비교

**결과물**: 시각화 그래프 3~4개, "SPY·QQQ는 양의 상관, TLT는 음의 상관" 같은 해석 기록

---

## 4회차 — 수익률 분석

**목표**: 일별/누적 수익률을 계산하고, 자산별 성과를 정량적으로 비교한다.

**구현 내용**

1. **일별 수익률 계산**
   ```python
   returns = price.pct_change().dropna()
   ```

2. **수익률 분포 히스토그램** — 각 자산의 수익률 분포 모양 비교 (정규분포에 가까운가?)

3. **누적 수익률 계산 및 시각화**
   ```python
   cumulative = (1 + returns).cumprod()
   ```

4. **연간 평균 수익률 계산**
   ```python
   annual_return = returns.mean() * 252
   ```

**결과물**: `returns` DataFrame, 누적 수익률 차트, 연간 수익률 요약 테이블

---

## 5회차 — 리스크 분석

**목표**: 수익률만으로는 알 수 없는 리스크를 수치화하고 자산별로 비교한다.

**구현 내용**

1. **연간 변동성 계산** — 일별 수익률 표준편차를 연간화
   ```python
   volatility = returns.std() * (252 ** 0.5)
   ```

2. **최대 낙폭 (MDD) 계산** — 고점 대비 최대 하락폭
   ```python
   rolling_max = cumulative.cummax()
   drawdown = (cumulative - rolling_max) / rolling_max
   mdd = drawdown.min()
   ```

3. **롤링 변동성 차트** — 252일 창으로 변동성의 시간 흐름 시각화

4. **리스크 요약 테이블** — 자산별 변동성 / MDD 한눈에 비교

**결과물**: MDD 차트, 롤링 변동성 차트, 리스크 비교 테이블

---

## 6회차 — 포트폴리오 전략 분석

**목표**: 3가지 전략을 설계하고 수익률·리스크 관점에서 비교한다.

**전략 설계**

| 전략 | 구성 | 비중 |
|------|------|------|
| 전략 A | QQQ 단일 | QQQ 100% |
| 전략 B | 주식 혼합 | SPY 50% + QQQ 50% |
| 전략 C | 주식+채권 | SPY 60% + TLT 40% |

**구현 내용**

1. **전략별 포트폴리오 수익률 계산**
   ```python
   weights_C = [0.6, 0.4]  # SPY, TLT
   portfolio_returns = (returns[["SPY", "TLT"]] * weights_C).sum(axis=1)
   ```

2. **전략별 누적 수익률 비교 차트**

3. **전략별 Sharpe Ratio 계산** (무위험 수익률 연 3% 가정)
   ```python
   sharpe = (annual_return - 0.03) / volatility
   ```

4. **전략 비교 테이블** — 수익률 / 변동성 / MDD / Sharpe 4가지 지표로 정리

**결과물**: 전략 비교 차트, 전략 성과 요약 테이블

## 7회차 — 포트폴리오 최적화

**목표**: 무수히 많은 비중 조합 중 효율적 포트폴리오를 찾고 Efficient Frontier를 시각화한다.

**구현 내용**

1. **Monte Carlo 시뮬레이션** — 랜덤 비중 10,000개 생성
   ```python
   weights = np.random.dirichlet(np.ones(3), size=10000)  # SPY, QQQ, TLT
   ```

2. **각 비중별 수익률·변동성·Sharpe 계산**

3. **Efficient Frontier 산점도** — x축: 변동성, y축: 수익률, 색상: Sharpe Ratio

4. **최적 포트폴리오 2개 식별**
   - Max Sharpe Portfolio (리스크 대비 수익 최대)
   - Min Volatility Portfolio (변동성 최소)

**결과물**: Efficient Frontier 시각화, 최적 비중 출력
